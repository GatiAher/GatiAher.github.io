# Add New Post

1. Create new post: `hugo new projects/<project-name>`

2. Fill in template

```
---
title: "My First Post"
date: 2019-03-26T08:47:11+01:00
draft: true
---
```

3. Start local server with drafts enabled: `hugo server -D`
```

4. Set `draft: False`

5. Deploy with `./deploy`
